Materi Algoritma dan Pemrograman

Pembelajaran Dasar Komputer untuk Siswa Kelas 8

Oleh : AAN SAPUTRA, S.Kom, Gr

1. Pengertian Algoritma

Algoritma adalah serangkaian langkah-langkah logis dan sistematis yang digunakan untuk menyelesaikan suatu masalah. Algoritma sangat penting dalam dunia komputer karena komputer hanya dapat bekerja jika diberikan perintah yang jelas dan terstruktur.

Dalam kehidupan sehari-hari sebenarnya kita sering menggunakan algoritma tanpa kita sadari. Misalnya ketika kita membuat mie instan, menyalakan komputer, atau melakukan perhitungan matematika.

Contoh Algoritma Membuat Teh

  1. Siapkan gelas
  2. Masukkan teh celup
  3. Tuangkan air panas
  4. Tambahkan gula
  5. Aduk hingga rata
  6. Teh siap diminum

Dari contoh tersebut kita dapat melihat bahwa setiap langkah harus dilakukan secara berurutan agar hasilnya sesuai dengan yang diharapkan.

2. Ciri-Ciri Algoritma

Sebuah algoritma yang baik memiliki beberapa ciri berikut:

  • Memiliki langkah-langkah yang jelas
  • Dapat diselesaikan dalam jumlah langkah terbatas
  • Memiliki input dan output
  • Setiap langkah mudah dipahami
  • Dapat diterapkan untuk menyelesaikan masalah

3. Pengertian Pemrograman

Pemrograman adalah proses membuat program komputer menggunakan bahasa pemrograman. Program komputer dibuat agar komputer dapat menjalankan suatu tugas tertentu secara otomatis.

Dalam pemrograman, seorang programmer harus mampu menyusun algoritma terlebih dahulu sebelum menulis kode program.

Contoh Bahasa Pemrograman

  • Python
  • Java
  • C++
  • JavaScript
  • PHP

Bahasa-bahasa tersebut digunakan untuk membuat berbagai aplikasi seperti website, aplikasi mobile, game, dan sistem komputer.

4. Pengenalan HTML

HTML adalah singkatan dari HyperText Markup Language. HTML digunakan untuk membuat struktur dasar sebuah halaman website.

HTML terdiri dari berbagai tag yang memiliki fungsi tertentu seperti membuat judul, paragraf, gambar, tabel, dan lain-lain.

Contoh Kode HTML

<h1>Selamat Datang</h1> <p>Ini adalah halaman website pertama saya</p>

5. Pengenalan CSS

CSS adalah singkatan dari Cascading Style Sheets. CSS digunakan untuk mengatur tampilan website agar lebih menarik.

Dengan CSS kita bisa mengubah warna teks, ukuran huruf, posisi elemen, dan berbagai desain lainnya.

Contoh Kode CSS

h1 { color: blue; font-size: 30px; }

6. Hubungan HTML dan CSS

HTML dan CSS biasanya digunakan secara bersamaan dalam pembuatan website. HTML berfungsi sebagai struktur halaman, sedangkan CSS berfungsi untuk mempercantik tampilan halaman tersebut.

Jika HTML diibaratkan sebagai kerangka rumah, maka CSS adalah cat dan dekorasi rumah tersebut.
